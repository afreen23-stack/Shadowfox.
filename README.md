# 📊 Data Visualization and Sentiment Analysis Project

## 📌 Overview

This repository contains two tasks focused on data analysis and visualization using Python. The project demonstrates the use of visualization libraries and natural language processing techniques to extract meaningful insights from data.

---

## 🧩 Task 1: Visualization Library Documentation

### 🔹 Description

This task explores the use of Python visualization libraries such as Matplotlib and Seaborn to create different types of graphs.

### 🔹 Work Done

* Created various plots:

  * Line Plot
  * Bar Chart
  * Scatter Plot
  * Histogram
  * Pie Chart
  * Box Plot
* Compared Matplotlib and Seaborn
* Understood data visualization techniques

---

## 💬 Task 2: Sentiment Analysis of X (Twitter) Data

### 🔹 Description

This task performs sentiment analysis on tweet data to understand public opinion using NLP concepts.

### 🔹 Work Done

* Processed tweet dataset
* Converted sentiment labels (-1, 0, 1 → Negative, Neutral, Positive)
* Performed analysis:

  * Sentiment Distribution (Bar Chart)
  * Sentiment Percentage (Pie Chart)
  * Word Cloud (Topic Analysis)
  * Text Length vs Sentiment (Pattern Analysis)

---

## ⚙️ Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn
* WordCloud

---

## 📊 Key Insights

* Positive sentiment dominates the dataset
* Neutral sentiment is significant
* Negative sentiment is comparatively lower
* Tweets focus on political and public topics
* Negative tweets tend to be longer
